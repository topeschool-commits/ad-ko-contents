<h2>@알기 쉬운 불교 이야기 여덟</h2><h3>― 사찰(寺刹)의 의미 ―</h3><hr><h3>1. 사찰(寺刹)의 명칭</h3><ul>
<li>
<p>사찰은 범어(梵語)로 가람(伽藍)이라 하며, 우리말로는 중원(衆院)이라 풀이됩니다.</p>
</li>
<li>
<p>중원이란 많은 대중이 모여 집회를 여는 것을 뜻합니다.</p>
</li>
<li>
<p>사찰의 순수 한글 표현은 ‘절’이며, 다음과 같이 불리기도 합니다:<br>
<strong>사찰(寺刹), 사원(寺院), 도량(道場), 정사(精舍)</strong></p>
</li>
<li>
<p>최초의 절은 부처님 당시의 죽림정사(竹林精舍)이고,<br>
두 번째 절 역시 부처님 당시의 기원정사(祇園精舍)입니다.</p>
</li>
<li>
<p>우리나라 최초의 절은 고구려 시대의 이불란사(伊弗蘭寺)입니다.</p>
</li>
</ul><hr><h3>2. 사찰의 구성</h3><ul>
<li>
<p>사찰의 건물 배치 구조는 다음과 같습니다:</p>
<ol>
<li>
<p><strong>일주문(一柱門)</strong></p>
<ul>
<li>
<p>정문 역할을 하며, 일승법(一乘法)을 의미합니다.</p>
</li>
<li>
<p>모든 진리가 하나로 돌아간다는 뜻입니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>천왕문(天王門)</strong></p>
<ul>
<li>
<p>사천왕신장(四天王神將)이 모셔져 있으며,<br>
부처님의 불법을 수호하는 신들의 공간입니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>불이문(不二門)</strong></p>
<ul>
<li>
<p>이 문을 지나면 불법의 세계에 들어가는 것으로,<br>
너와 나, 삶과 죽음, 무와 유가 둘이 아님을 의미합니다.</p>
</li>
<li>
<p>하나의 <strong>불법세계</strong>로 진입하게 됩니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>법당(法堂)</strong></p>
<ul>
<li>
<p>사찰의 중심입니다.</p>
</li>
<li>
<p>본 법당 외에 다른 법당이 있을 수도 있으며,<br>
법당 주변으로 <strong>탑</strong>이 안치되고 <strong>요사채</strong>가 배치됩니다.</p>
</li>
</ul>
</li>
</ol>
</li>
<li>
<p>이처럼 사찰은 <strong>법당, 문루(門樓), 탑(塔), 요사(寮舍)</strong> 등이 하나의 건물군으로 조화되어<br>
<strong>사찰의 형태</strong>를 이룹니다.</p>
</li>
</ul><hr><h3>3. 사찰의 상징성</h3><ul>
<li>
<p>큰 사찰은 <strong>대웅전에서 일주문까지</strong> 크고 작은 법당, 요사, 탑, 등, 종 등이 함께 어우러져<br>
입체적으로 하나의 <strong>만다라(曼茶羅)</strong> 현상을 나타냅니다.</p>
</li>
<li>
<p>사찰은 <strong>극락세계</strong>를 표현한 장소이자,<br>
입체적 만다라이며 <strong>극락세계의 상징</strong>입니다.</p>
</li>
</ul><hr><p><b>ㅡ대한불교조계종 포교사 ㆍ명궁작명철학관장 자향법사 심 재민((010ㅡ3328ㅡ2729))합장 삼배 ㅡ</b>
