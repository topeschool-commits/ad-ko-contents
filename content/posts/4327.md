<h3><strong>알기쉬운 불교이야기 둘</strong></h3><h4><strong>1. 불교의 우주관</strong></h4><p>불교의 우주관은 <strong>삼법인</strong>(三法印)입니다. 삼법인은 '연기법을 설명한 확실한 진리'라는 뜻입니다. 삼법인은 <strong>제행무상</strong>, <strong>제법무아</strong>, <strong>열반적정</strong> 세 가지로 나누어집니다.</p><ul>
<li>
<p><strong>제행무상</strong> (諸行無常)</p>
<ul>
<li>
<p>모든 현상은 인연에 따라 왔다가 인연이 다하면 사라지는 덧없는 것이라 말합니다. 석가모니 부처님께서는 열반하시기 직전에 "모든 것은 변한다"라고 제행무상(諸行無常)을 설하셨습니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>제법무아</strong> (諸法無我)</p>
<ul>
<li>
<p>모든 존재나 현상에는 실질적인 본질이 없으며, 어떤 환경에 의해 생겼다가 다른 환경에 의해 사라지기 때문에 실체가 없다는 무아론(無我論)입니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>열반적정</strong> (涅槃寂靜)</p>
<ul>
<li>
<p>근본 진리는 <strong>유</strong>와 <strong>무</strong>를 초월하고 <strong>무상</strong>과 <strong>무아</strong>를 넘어서서 늘 고요하고 실상 그대로 존재하는 것입니다.</p>
</li>
</ul>
</li>
</ul><h4><strong>2. 불교의 인생관</strong></h4><p>불교의 인생관이자 생활관은 <strong>사성제</strong>(四聖諦)입니다. 사성제는 <strong>고</strong>(苦), <strong>집</strong>(集), <strong>멸</strong>(滅), <strong>도</strong>(道) 네 가지를 의미합니다.</p><ul>
<li>
<p><strong>고성제</strong> (苦聖諦)</p>
<ul>
<li>
<p>인간에게 주어지는 생(生), 노(老), 병(病), 사(死) 등 육체적 고통과 불안, 좌절, 분노, 고독 등 정신적 고통을 말합니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>집성제</strong> (集聖諦)</p>
<ul>
<li>
<p>인간의 괴로움의 원인은 번뇌, 망상에서 오며, 이들은 탐(貪), 진(嗔), 치(痴) 삼독심(三毒心)에서 비롯됩니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>멸성제</strong> (滅聖諦)</p>
<ul>
<li>
<p>번뇌와 고통이 모두 없어진 해탈의 세계, 열반의 세계, 이상의 세계를 말합니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>도성제</strong> (道聖諦)</p>
<ul>
<li>
<p>고통과 괴로움에서 벗어나 해탈 및 열반에 이를 수 있는 실천행위와 수도론(修道論), 실천론(實踐論)입니다. 이는 고행이나 즐김이 아닌, <strong>불고불락</strong>(不苦不樂)의 <strong>중도적 실천</strong>을 통해 열반과 해탈의 세계로 갈 수 있습니다.</p>
</li>
</ul>
</li>
</ul><h4><strong>3. 중도 실천으로서의 계(戒), 정(定), 혜(慧) 삼학(三學)</strong></h4><p><strong>삼학</strong>(三學)은 중도 실천을 위한 중요한 요소입니다.</p><ul>
<li>
<p><strong>계</strong> (戒)</p>
<ul>
<li>
<p>악한 일을 멀리하기 위해 자기 자신을 규제하는 것입니다. 즉, 모든 악을 추방하여 화합하고 자비로운 방향으로 나가자는 의미입니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>정</strong> (定)</p>
<ul>
<li>
<p>항상 자기 자신을 되돌아보고 반성하여 마음을 정화시켜 나가는 것입니다.</p>
</li>
</ul>
</li>
<li>
<p><strong>혜</strong> (慧)</p>
<ul>
<li>
<p>모든 좋은 일과 창조적인 일을 적극적으로 실천하는 것입니다.</p>
</li>
</ul>
</li>
</ul><h4><strong>4. 삼학의 기본정신</strong></h4><ul>
<li>
<p><strong>계</strong> (戒): 사람의 탐욕을 제어하기 위한 계율입니다.</p>
</li>
<li>
<p><strong>정</strong> (定): 성냄과 분노를 가라앉히기 위한 선정입니다.</p>
</li>
<li>
<p><strong>혜</strong> (慧): 무지와 어리석음을 바로잡기 위한 창조적인 선행인 지혜입니다.</p>
</li>
