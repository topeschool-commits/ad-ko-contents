<h3><strong>비소포성 림프종 (C83)</strong></h3><p><b><span style="font-size: 14pt;">비소포성 림프종(Non-Follicular Lymphoma)</span></b>은 국제질병분류(ICD-10) 코드 C83으로 분류되며, 비호지킨 림프종(Non-Hodgkin Lymphoma, NHL)의 하위 유형 중 하나입니다. 이는 림프구(B세포 또는 T세포)의 악성 증식으로 인해 발생하며, 림프절 외에도 신체의 다양한 부위에서 발생할 수 있습니다.</p><p>비소포성 림프종은 종양이 소포(follicle)를 형성하지 않는 것이 특징이며, 일반적으로 빠르게 진행되는 고등급 림프종으로 분류됩니다. 이는 적극적인 치료가 필요한 경우가 많습니다.</p><hr><h4><strong>1. 비소포성 림프종의 유형</strong></h4><p>비소포성 림프종은 조직학적 특성과 원인에 따라 여러 아형으로 나뉩니다.</p><ol>
<li>
<p><strong>미만성 거대 B세포 림프종(Diffuse Large B-Cell Lymphoma, DLBCL)</strong></p>
<ul>
<li>가장 흔한 비소포성 림프종으로 전체 비호지킨 림프종의 약 30% 차지.</li>
<li>빠르게 진행하며 적극적인 치료가 필요.</li>
</ul>
</li>
<li>
<p><strong>말초 T세포 림프종(Peripheral T-Cell Lymphoma, PTCL)</strong></p>
<ul>
<li>T세포에서 기원하며 예후가 다소 불량한 편.</li>
</ul>
</li>
<li>
<p><strong>버킷 림프종(Burkitt Lymphoma)</strong></p>
<ul>
<li>매우 빠르게 증식하며 주로 소아와 젊은 성인에게 발생.</li>
<li>Epstein-Barr 바이러스와 관련.</li>
</ul>
</li>
<li>
<p><strong>역형성 대세포 림프종(Anaplastic Large Cell Lymphoma, ALCL)</strong></p>
<ul>
<li>T세포 림프종의 한 형태로, 특정 유전자 변이(ALK 양성 또는 음성)에 따라 나뉨.</li>
</ul>
</li>
</ol><hr><h4><strong>2. 주요 증상</strong></h4><p>비소포성 림프종의 증상은 병변의 위치와 진행 정도에 따라 다양합니다.</p><ol>
<li>
<p><strong>림프절 증상</strong></p>
<ul>
<li><strong>무통성 림프절 비대</strong>: 목, 겨드랑이, 사타구니 등에서 흔히 관찰.</li>
<li>종양이 빠르게 커지는 특징.</li>
</ul>
</li>
<li>
<p><strong>전신 증상(B 증상)</strong></p>
<ul>
<li>고열</li>
<li>야간 발한</li>
<li>원인 불명의 체중 감소</li>
</ul>
</li>
<li>
<p><strong>기타 증상</strong></p>
<ul>
<li>복부 통증 및 팽만: 복부 림프절 침범 시 발생.</li>
<li>호흡 곤란 또는 흉통: 흉부 림프절이 확대된 경우.</li>
<li>피부 병변: 피부 림프종에서 발생 가능.</li>
</ul>
</li>
</ol><hr><h4><strong>3. 원인 및 위험 요인</strong></h4><p>비소포성 림프종의 원인은 명확하지 않지만, 다음과 같은 요인이 관련될 수 있습니다:</p><ul>
<li><strong>유전적 변이</strong>: 종양 억제 유전자의 결함 또는 특정 염색체 전위.</li>
<li><strong>바이러스 감염</strong>: Epstein-Barr 바이러스(EBV), 인간 T세포 림프친화성 바이러스(HTLV-1) 등.</li>
<li><strong>면역 체계 약화</strong>: HIV 감염, 면역억제제 사용.</li>
<li><strong>화학 물질 노출</strong>: 농약 및 특정 산업용 화학 물질.</li>
<li><strong>자가면역 질환</strong>: 루푸스, 쇼그렌 증후군 등.</li>
</ul><hr><h4><strong>4. 진단 과정</strong></h4><p>비소포성 림프종의 진단은 다음의 검사를 통해 이루어집니다:</p><ol>
<li>
<p><strong>신체검사</strong></p>
<ul>
<li>림프절 비대 및 기타 증상 확인.</li>
</ul>
</li>
<li>
<p><strong>조직 생검</strong></p>
<ul>
<li>비정상 림프절 또는 조직을 채취하여 병리학적 분석.</li>
</ul>
</li>
<li>
<p><strong>영상 검사</strong></p>
<ul>
<li><strong>CT 스캔</strong> 및 <strong>PET 스캔</strong>: 종양의 크기와 전이 범위 평가.</li>
<li><strong>MRI</strong>: 중추신경계 림프종 의심 시 유용.</li>
</ul>
</li>
<li>
<p><strong>혈액 검사</strong></p>
<ul>
<li>전반적인 혈구 상태, LDH 수치, 특정 바이러스 감염 여부 확인.</li>
</ul>
</li>
<li>
<p><strong>골수 검사</strong></p>
<ul>
<li>골수 침범 여부 확인.</li>
</ul>
</li>
<li>
<p><strong>유전자 검사</strong></p>
<ul>
<li>특정 유전자 변이 및 염색체 이상 확인.</li>
</ul>
</li>
</ol><hr><h4><strong>5. 치료 방법</strong></h4><p>비소포성 림프종의 치료는 병기, 세포 유형, 환자의 전반적인 건강 상태에 따라 결정됩니다.</p><ol>
<li>
<p><strong>항암화학요법</strong></p>
<ul>
<li>R-CHOP 요법(리툭시맙, 시클로포스파마이드, 독소루비신, 빈크리스틴, 프레드니손) 등.</li>
</ul>
</li>
<li>
<p><strong>방사선 치료</strong></p>
<ul>
<li>국소 부위에 제한된 림프종에 사용.</li>
</ul>
</li>
<li>
<p><strong>표적 치료</strong></p>
<ul>
<li><strong>리툭시맙</strong>: CD20 단백질을 표적으로 하는 B세포 림프종 치료제.</li>
<li><strong>CAR-T 세포 치료</strong>: 환자의 면역 세포를 변형하여 암세포 제거.</li>
</ul>
</li>
<li>
<p><strong>조혈모세포 이식</strong></p>
<ul>
<li>재발성 또는 치료 저항성이 있는 경우 고용량 화학요법 후 시행.</li>
</ul>
</li>
<li>
<p><strong>임상시험 및 신약</strong></p>
<ul>
<li>새로운 약물이나 치료법에 참여 가능.</li>
</ul>
</li>
</ol><hr><h4><strong>6. 예후 및 관리</strong></h4><ul>
<li>비소포성 림프종의 예후는 아형, 병기, 환자의 나이에 따라 달라집니다.</li>
<li>미만성 거대 B세포 림프종은 치료에 잘 반응하는 편이며, 조기 발견 시 완치 가능성이 높습니다.</li>
<li>정기적인 추적 검사를 통해 재발 여부를 모니터링해야 합니다.</li>
</ul><hr><p><strong><br></strong></p><p><strong>위의 내용은 일반적인 참고 자료로 제공된 것이며, 개별적인 상황에 맞는 정확한 진단과 치료를 위해서는 반드시 전문의와 상담하시기 바랍니다.</strong></p>
