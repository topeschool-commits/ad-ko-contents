<h3><strong>호지킨 림프종 (C81)</strong></h3><p><b><span style="font-size: 14pt;">호지킨 림프종(Hodgkin Lymphoma)</span></b>은 국제질병분류(ICD-10) 코드 C81로 분류되며, 림프계에 발생하는 악성 종양의 한 유형입니다. 림프계는 면역 체계의 일부로서 림프절, 림프관, 비장, 골수 등을 포함하며 체내에서 감염과 질병을 방어하는 역할을 합니다.</p><p>호지킨 림프종은 림프구(특히 B세포)의 비정상적인 성장을 특징으로 하며, 다른 조직과 기관으로 퍼질 수 있습니다. 다른 림프종과 구별되는 특징은 <strong>Reed-Sternberg 세포</strong>라는 특정 비정상 세포가 발견된다는 점입니다.</p><hr><h4><strong>1. 호지킨 림프종의 유형</strong></h4><p>호지킨 림프종은 두 가지 주요 유형으로 나뉩니다.</p><ol>
<li>
<p><strong>고전적 호지킨 림프종(Classic Hodgkin Lymphoma)</strong></p>
<ul>
<li>전체 호지킨 림프종의 약 90%를 차지하며, Reed-Sternberg 세포가 특징적입니다.</li>
<li>4가지 아형으로 구분됩니다:
<ul>
<li>결절경화형(Nodular Sclerosis): 가장 흔한 유형</li>
<li>혼합세포형(Mixed Cellularity): 노년층에서 주로 발생</li>
<li>림프구 풍부형(Lymphocyte Rich): 예후가 비교적 좋음</li>
<li>림프구 소실형(Lymphocyte Depleted): 드물고 공격적인 형태</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>결절림프구우세 호지킨 림프종(Nodular Lymphocyte-Predominant Hodgkin Lymphoma)</strong></p>
<ul>
<li>비교적 드문 유형으로, Reed-Sternberg 세포 대신 "팝콘 세포"라고 불리는 비정상 세포가 관찰됩니다.</li>
<li>성장 속도가 느리고 예후가 좋은 편입니다.</li>
</ul>
</li>
</ol><hr><h4><strong>2. 주요 증상</strong></h4><p>호지킨 림프종의 증상은 림프절이 비대해지거나 종양이 위치한 부위에 따라 다양합니다.</p><ol>
<li>
<p><strong>림프절 증상</strong></p>
<ul>
<li><strong>무통성 림프절 비대</strong>: 주로 목, 겨드랑이, 사타구니 부위에서 발생</li>
<li>비대해진 림프절은 음주 시 통증을 유발할 수 있음</li>
</ul>
</li>
<li>
<p><strong>전신 증상(B 증상)</strong></p>
<ul>
<li>고열(원인 불명의 지속적인 열)</li>
<li>야간 발한(심한 땀을 흘림)</li>
<li>체중 감소(6개월 이내 10% 이상)</li>
</ul>
</li>
<li>
<p><strong>기타 증상</strong></p>
<ul>
<li>만성 피로</li>
<li>피부 가려움증</li>
<li>기침, 호흡곤란: 흉부 림프절이 압박될 경우 발생</li>
<li>복부 팽만 및 통증</li>
</ul>
</li>
</ol><hr><h4><strong>3. 원인 및 위험 요인</strong></h4><p>호지킨 림프종의 정확한 원인은 알려져 있지 않지만, 다음과 같은 요인이 관련될 수 있습니다:</p><ul>
<li><strong>바이러스 감염</strong>: Epstein-Barr 바이러스(EBV)와 연관성이 높음</li>
<li><strong>면역 체계 약화</strong>: HIV 감염자나 면역억제제를 사용하는 사람들</li>
<li><strong>가족력</strong>: 림프종을 가진 가족이 있는 경우 위험 증가</li>
<li><strong>연령</strong>: 15~30세와 55세 이상의 연령대에서 발병률이 높음</li>
<li><strong>성별</strong>: 남성이 여성보다 발병률이 약간 높음</li>
</ul><hr><h4><strong>4. 진단 과정</strong></h4><p>호지킨 림프종은 다음의 과정을 통해 진단됩니다:</p><ol>
<li>
<p><strong>신체검사</strong></p>
<ul>
<li>비정상적인 림프절 비대 여부 및 관련 증상 확인</li>
</ul>
</li>
<li>
<p><strong>림프절 생검</strong></p>
<ul>
<li>림프절 조직을 채취하여 Reed-Sternberg 세포의 유무 확인</li>
</ul>
</li>
<li>
<p><strong>혈액 검사</strong></p>
<ul>
<li>백혈구, 적혈구, 혈소판 수치 평가</li>
<li>LDH(젖산탈수소효소) 수치 상승 여부 확인</li>
</ul>
</li>
<li>
<p><strong>영상 검사</strong></p>
<ul>
<li><strong>CT 스캔</strong> 및 <strong>PET 스캔</strong>: 종양의 크기와 위치, 전이 여부 파악</li>
<li><strong>MRI</strong>: 특정 부위의 자세한 영상 제공</li>
</ul>
</li>
<li>
<p><strong>골수 검사</strong></p>
<ul>
<li>골수 침범 여부 확인</li>
</ul>
</li>
</ol><hr><h4><strong>5. 치료 방법</strong></h4><p>호지킨 림프종의 치료는 병기, 환자의 나이와 건강 상태에 따라 결정됩니다.</p><ol>
<li>
<p><strong>항암화학요법</strong></p>
<ul>
<li>ABVD(아드리아마이신, 블레오마이신, 빈블라스틴, 다카르바진) 등 다약제 요법이 일반적으로 사용됩니다.</li>
</ul>
</li>
<li>
<p><strong>방사선 치료</strong></p>
<ul>
<li>초기 병기의 경우 림프절 부위에 방사선을 조사하여 치료 효과를 높임</li>
</ul>
</li>
<li>
<p><strong>표적 치료</strong></p>
<ul>
<li>특정 단백질이나 세포를 표적으로 하여 종양 세포를 제거</li>
</ul>
</li>
<li>
<p><strong>면역요법</strong></p>
<ul>
<li>면역 체계를 활성화하여 암세포와 싸움</li>
</ul>
</li>
<li>
<p><strong>조혈모세포 이식</strong></p>
<ul>
<li>재발성 또는 치료에 반응하지 않는 경우 시행</li>
</ul>
</li>
</ol><hr><h4><strong>6. 예후 및 관리</strong></h4><ul>
<li>호지킨 림프종은 치료 가능한 암 중 하나로 간주되며, 조기 발견 시 완치율이 높습니다.</li>
<li>치료 후 정기적인 추적 검사를 통해 재발 여부를 확인해야 합니다.</li>
<li>건강한 생활 습관과 면역력 강화는 재발 위험을 줄이는 데 중요합니다.</li>
</ul><hr><p><strong><br></strong></p><p><strong>위의 내용은 일반적인 참고 자료로 제공된 것이며, 개별적인 상황에 맞는 정확한 진단과 치료를 위해서는 반드시 전문의와 상담하시기 바랍니다.</strong></p>
