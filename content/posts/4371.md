<h2>1. 언어(모국어) 및 의사소통</h2>
<ul>
<li>
<p>왜 중요한가요?</p>
<ul>
<li>
<p>모든 직업은 사람과의 소통을 전제로 합니다.</p>
</li>
<li>
<p>문서를 읽고 쓰거나, 보고하고 설득하는 능력은 직무의 기본입니다.</p>
</li>
</ul>
</li>
<li>
<p>포함되는 능력</p>
<ul>
<li>
<p>글쓰기, 말하기, 읽기, 듣기</p>
</li>
<li>
<p>논리적 사고, 설득력, 발표력</p>
</li>
</ul>
</li>
</ul>
<hr>
<h2>2. 수학(기초 연산 및 논리적 사고)</h2>
<ul>
<li>
<p>왜 중요한가요?</p>
<ul>
<li>
<p>단순히 숫자를 다루는 직업뿐 아니라, 문제를 논리적으로 해결하는 능력의 기반이 됩니다.</p>
</li>
<li>
<p>데이터 해석, 비용 계산, 계획 수립 등 거의 모든 직종에서 필요합니다.</p>
</li>
</ul>
</li>
<li>
<p>포함되는 능력</p>
<ul>
<li>
<p>기초 연산, 분석적 사고, 논리적 추론, 문제 해결</p>
</li>
</ul>
</li>
</ul>
<hr>
<h2>3. 과학적 사고(기초 과학 원리)</h2>
<ul>
<li>
<p>왜 중요한가요?</p>
<ul>
<li>
<p>세상을 객관적으로 바라보고, 원인을 분석하며, 증거를 기반으로 판단하는 힘을 줍니다.</p>
</li>
<li>
<p>특히 기술, 의학, 엔지니어링, 농업 등 다양한 분야의 기초가 됩니다.</p>
</li>
</ul>
</li>
<li>
<p>포함되는 능력</p>
<ul>
<li>
<p>실험적 사고, 원리 이해, 비판적 사고</p>
</li>
</ul>
</li>
</ul>
<hr>
<h2>4. 사회 및 윤리(시민의식, 도덕)</h2>
<ul>
<li>
<p>왜 중요한가요?</p>
<ul>
<li>
<p>사회적 규범과 타인과의 협력, 책임감을 배우는 과목입니다.</p>
</li>
<li>
<p>조직과 사회 속에서 올바르게 행동하고, 윤리적 판단을 할 수 있는 기반이 됩니다.</p>
</li>
</ul>
</li>
<li>
<p>포함되는 능력</p>
<ul>
<li>
<p>사회 이해, 법과 제도, 도덕성, 협력, 공동체 의식</p>
</li>
</ul>
</li>
</ul>
<hr>
<h3>종합적으로 말하면</h3>
<p>모국어(언어), 수학, 과학, 사회 및 윤리 이 네 가지가 모든 직업의 공통된 기초 과목이라고 할 수 있습니다.</p>
