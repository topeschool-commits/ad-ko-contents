@알기쉬운 불교이야기 셋

ㅡ 불교의 종파와 육바라밀 ㅡ

1. 불교의 종파(宗派)

불교의 종파는 다른 종교처럼 여러 종파로 나누어져 있으나,
크게 교종(敎宗)과 선종(禪宗),
소승불교(小乘佛敎)와 대승불교(大乘佛敎)로 분류할 수 있다.

2. 교종(敎宗)과 선종(禪宗)

  • 교종은 부처님 경전(經典)을 중심으로 수행하고 연구해 나가는 종파

  • 선종은 부처님 경전보다 화두(話頭) 참선(參禪) 위주로 수행하는 종파

3. 소승불교 vs 대승불교

  • 소승불교(小乘佛敎)
    → 인간의 욕망 자체를 반성하고 다스리는 것으로 해탈을 추구
    → 출가수도(出家)를 권장
    → 이욕해탈(離欲解脫)의 불교

  • 대승불교(大乘佛敎)
    → 자비를 실천함으로써 욕망을 초월하고 해탈을 성취
    → 자비해탈(慈悲解脫)의 종교
    → 상구보리(上求菩提)와 하화중생(下化衆生)을 실천
    → 진리를 사랑하고 인간을 사랑하는 보살(菩薩)을 이상으로 함

4. 육바라밀(六波羅蜜)

대승불교의 핵심 수행법

  • 보시(布施): 베풀어주는 것

  • 지계(持戒): 계율을 잘 실천하는 것

  • 인욕(忍辱): 욕됨을 잘 참는 것

  • 선정(禪定): 마음을 안정시키는 것

  • 정진(精進): 항상 노력하는 것

  • 지혜(智慧): 좋은 일을 실천하는 것

삼학(三學) – 계(戒), 정(定), 혜(慧)
→ 이 삼학의 대승불교적 실천이 바로 육바라밀

  • 지계와 인욕: 계(戒)

  • 정진과 선정: 정(定)

  • 보시와 지혜: 혜(慧)

→ 깨달음을 향한 실천방법은 연기와 중도, 그리고 그 실천이 삼학과 육바라밀이다.
